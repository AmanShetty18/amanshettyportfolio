My Work

Custom Craft

Custom Craft is an e-commerce website designed to simplify the process of purchasing laptops, computers, software, and hardware. Our platform helps users find the perfect tech solutions tailored to their specific needs. With a user-friendly interface and curated product selections, Custom Craft makes it easy for everyone, from students to professionals, to discover the right technology for their requirements.

Custom Craft is built using HTML, CSS, and JavaScript for the front end, providing a responsive and interactive user experience. The back end is powered by PHP, which manages the database for seamless product management and user transactions. This combination of technologies ensures a robust and efficient e-commerce platform that meets users' needs.

Game Design

I developed Sherlock, a 2D interactive game created using the Unity game engine. This game features an engaging dialogue between two characters, Sherlock and John, as they collaboratively search for John’s misplaced belongings scattered throughout the room. The immersive gameplay combines problem-solving and narrative elements to enhance the user experience.
